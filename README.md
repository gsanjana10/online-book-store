# Online Book Store 📚

## Project Description
A simple Online Book Store website built using HTML, CSS, JavaScript, Node.js, and Express.

## Features
- Display books
- Show prices
- Add to Cart button
- Backend API using Express

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## API Endpoint

http://localhost:5000/books
